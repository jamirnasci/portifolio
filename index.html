Meus Projetos

Forex Track

o Forex Track oferece uma ampla gama de funcionalidades, incluindo análise detalhada de pares de moedas, além dos indicadores essenciais RSI, Estocástico, MACD e Médias Móveis. Com uma interface intuitiva e atualizações em tempo real, o Forex Track é a ferramenta para auxiliar operações no forex fornecendo possíveis sinais de compra ou venda de acordo com o modelo matemático de cada indicador.

Prodigal Web

Uma versão web do Preditor Prodigal, uma ferramenta que desenvolvi para facilitar a análise de sequências genômicas. Com esta nova versão, os usuários agora podem acessar o Prodigal diretamente de seus navegadores, sem a necessidade de instalação ou configuração de software adicional.
🔬 O que é o Prodigal? Prodigal é um preditor de genes procarióticos amplamente utilizado, conhecido por sua precisão e eficiência na identificação de genes em sequências de DNA. Ele é uma ferramenta essencial para pesquisadores em bioinformática e genômica.

Evaluating the use of smartphones for processing computational tasks: "A case study for the annotation of prokaryotic genomes"

Resultados de um projeto de pesquisa, onde exploramos o potencial do Android para executar ferramentas de anotação de genomas procariontes. Utilizando Termux e Docker, conseguimos viabilizar a aplicação do Prokka, abrindo portas para uma análise mais acessível e eficiente.

Android Alignment Tool

O Android Alignment Tool é uma ferramenta criada para android com intuito de fazer o alinhamento global de sequências biológicas com base no algoritmo de Needleman-Wunsch, além disso o app fornece uma análise estatística sobre os resultados, calculando gaps, matches e mismatches.
